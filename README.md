# NightOwl
Project managment application to manage your "precious" projects

# Getting Started

## Developers
Make Sure you `Cargo`. To get a copy of the source code for development:
```shell
$ git clone https://github.com/alialboainin96/Nightowl.git
```
Compiling code:
```shell
$ cargo run
```
